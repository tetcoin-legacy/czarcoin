18 // Run some of the tests registered by the TEST() macro. If the
19 // environment variable "LEVELDB_TESTS" is not set, runs all tests.

21 // "LEVELDB_TESTS" as a substring. E.g., suppose the tests are:
22 // TEST(Foo, Hello) { ... }
